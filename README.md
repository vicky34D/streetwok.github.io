## üåê How to Deploy to GitHub Pages

This project is ready to be hosted for free on GitHub Pages!

1.  **Create a Repository**:
    *   Go to [GitHub.com](https://github.com) and create a new repository (e.g., `streetwok-website`).
    *   Do **not** initialize with a README, .gitignore, or license (since you already have files).

2.  **Upload Files**:
    *   Open your terminal in this project folder.
    *   Run the following commands:
        ```bash
        git init
        git add .
        git commit -m "Initial commit - StreetWok Website"
        git branch -M main
        git remote add origin https://github.com/YOUR_USERNAME/streetwok-website.git
        git push -u origin main
        ```
    *   *(Replace `YOUR_USERNAME` and `streetwok-website` with your actual details)*.

3.  **Activate GitHub Pages**:
    *   Go to your repository **Settings** on GitHub.
    *   Click on **Pages** in the left sidebar.
    *   Under **Source**, select `Deploy from a branch`.
    *   Under **Branch**, select `main` and `/ (root)`.
    *   Click **Save**.

4.  **Done!**:
    *   Wait a minute or two. GitHub will give you a link (e.g., `https://yourusername.github.io/streetwok-website/`).
    *   Click it to see your live site!
